# Nova Coaching – Landing Page

Landing page moderne et immersive pour un coach sportif **NOVA Coaching**, avec :

- fond animé en **3D** (Three.js)
- effets de **parallaxe** sur la carte principale
- sections claires : Objectifs, Coaching, Progression, Réservation

## Structure

- `index.html` – structure de la page
- `styles.css` – design, animations et responsive
- `main.js` – canvas 3D, effets d’animation et interactions

## Utilisation

1. Ouvre simplement le fichier `index.html` dans ton navigateur (double‑clic).
2. Pour modifier les textes, édite directement `index.html`.
3. Pour ajuster les couleurs / animations, passe par `styles.css` et `main.js`.

## Logo / image du coach

Place ton image (logo + coach) dans un dossier `assets/` à la racine de ce projet (par exemple `assets/nova-hero.png`) puis utilise‑la dans `index.html` ou en fond de section via CSS.
